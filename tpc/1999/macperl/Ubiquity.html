Ubiquity: "A Perl in Every Process"

  • Perl support increasingly attractive for various applications (editors, IDEs, WWW servers).
  • Current distribution of MacPerl not very embedding friendly.
  • Plan to support a MacPerl library (dynamic if possible).
  • Clients linking to library can execute arbitrary number of concurrent Perl programs.
  • Possibility of faceless Perl server, reducing startup times for lightweight perl scripts.
