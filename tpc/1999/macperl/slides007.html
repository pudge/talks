Perl vs. AppleScript

  • MacPerl usually executes more quickly, though AppleScripts can be precompiled
  • Embed AppleScript with MacPerl::DoAppleScript built-in function
  • Embed Perl in AppleScript with MacPerl's Do Script event
  • If you know Perl, Perl development is mostly faster and easier. If not, AppleScript may be faster and easier. In any case, developing scripts with Apple events is faster and easier in AppleScript ... maybe.
