More Cool Stuff

  • Mac::Glue automatically guesses (with help from info in the aete resource) what data types elements should be: aliases converted to alias data with Mac::Files::NewAliasMinimal, booleans converted, etc. 
        @files = $sherlock->search('Bird:', For => 'MacOS');
  • Mac::Glue recursively converts arrays and hashes to AE lists and records (and on output, lists and records to arrays and hashes) 
        $glue->make(new => window => with_properties =>
        {name => 'Window', position => [100, 300]}
    );
