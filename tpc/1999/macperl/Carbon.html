Carbon Perl Support

  • Carbon makes large subset of MacOS 8/9 toolbox available to MacOS X applications.
  • Current MacPerl toolbox interface libraries largely portable to Carbon.
  • Entirely independent of Perl core.
  • Some demo Perl/Carbon programs already run.
  • Improved memory management and process scheduling in MacOS X promises increasing role for Perl based UI applications.
