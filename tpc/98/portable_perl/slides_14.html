Issue #8: System Resources

Be mindful of open file limitations.

Some platforms have no, or limited, virtual memory schemes. Be especially mindful of this when writing portable Perl:

  # NOTE: this is no longer "bad" in perl5.005
  for (0..10000000) { ... }                      # BAD
  for (my $x = 0; $x <= 10000000; ++$x) { ... }  # GOOD

  @lines = <VERY_LARGE_FILE>;                    # BAD

  while (<FILE>) {$file .= $_}                   # BAD
  $file = join('', <FILE>);                      # GOOD
