Issue #7: Time and Date

Unix epoch starts at January 01, 1970, 00:00:00 UTC.

This is not a universally supported standard.

Store dates in an unambiguous representation.

  • An array of values to pass to Time::Local::timelocal()
  • A text string (e.g., 01 Jan 1970) to pass to a module like Date::Parse
