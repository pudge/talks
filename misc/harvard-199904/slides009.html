Issue #3: System Interaction ("Don't"s Part I)

  • No command line? Don't worry about it, but realize it

  • Don't depend on %ENV

  • Don't use %SIG

  • Don't glob (use readdir instead)

  • Don't rely on value of $!

  • No UID, no GID, no HOME
